# Age Calculator

A simple web application that calculates your age based on the date of birth you enter. Built using **JavaScript**, **HTML**, and **CSS**.

## Features
- **Interactive Interface:** Users can input their date of birth.
- **Age Calculation:** Displays the user's age in years based on the provided birthdate.
- **Responsive Design:** Works on all screen sizes.

## Technologies Used
- **JavaScript:** To calculate the age dynamically.
- **HTML:** To structure the page.
- **CSS:** For styling the page with a clean and modern design.

## How to Use
1. Enter your **date of birth** in the input field.
2. Click the **"Calculate Age"** button.
3. View your calculated age displayed below the button.
